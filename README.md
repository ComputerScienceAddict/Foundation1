# Awesome Foundation Website

A React + Next.js website for **Awesome Foundation**—a 501(c)(3) nonprofit supporting cardiac patients and families.

## Features

- **Home** – Mission, quick links, and contact
- **Resources** – Curated resources for support groups, financial assistance, and medical information
- **Share Your Journey** – Form for families to share their story
- **Heart Disease Info** – Information about congenital heart defects and acquired heart conditions

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4

## Getting Started

```bash
cd Resource-website
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) (runs on port 3001 to avoid conflict with parent project).

## Build

```bash
npm run build
npm run start
```
